# Invest Content Studio (インベスト・コンテンツ・スタジオ)

AIを活用した投資関連のテキスト記事生成から、将来的には音声（ポッドキャスト・ナレーション）や動画（解説動画・スライドショー）までを一貫して自動生成するシステムです。

## 🚀 プロジェクトの概要

本プロジェクトは、日々変化する金融・投資情報を収集・分析し、ユーザーにとって価値のある投資情報を以下のマルチメディア形式で自動生成することを目指します。

1. **Text (記事)**: 最新の市場データやニュースを元に、LLM（Gemini等）を活用して高品質な解説記事を生成。
2. **Audio (音声)**: 生成された記事をベースに、自然な音声合成（TTS）技術を用いて音声解説（音声モジュール）を作成。
3. **Video (動画)**: 記事と音声を統合し、スライドやグラフ、アバターを用いた解説動画を自動合成。

## 🛠️ アーキテクチャ設計 (Monorepo構成)

将来的な機能拡張（Web UI、複数の生成エンジン、API等）を円滑に行うため、**Monorepo（モノリポ）** 構成を採用します。

```text
invest-content-studio/
├── apps/
│   └── web/                # 管理・プレビュー用Webダッシュボード (Next.js / React)
├── packages/
│   ├── article-generator/  # 投資記事生成エンジン (Python または TypeScript)
│   │                       #  - データソースのスクレイピング/API連携
│   │                       #  - LLMによる記事生成ロジック
│   ├── audio-generator/    # 音声合成エンジン (TTS連携)
│   ├── video-generator/    # 動画合成エンジン (FFmpeg / 自動レンダリング)
│   └── shared/             # 共通の型定義、共通ユーティリティ
├── README.md               # プロジェクト概要
└── .gitignore              # Git無視ファイル設定
```

## 📅 開発ロードマップ

### Phase 1: 記事生成エンジン (Text) の構築
* 金融データ（株価、ニュース、適時開示等）のクローリングまたはAPI連携の仕組み作り。
* Gemini API等のLLMを使用した、投資記事の自動ドラフト作成エンジン。
* マークダウン形式での出力と、SEOを意識した構成テンプレートの設計。

### Phase 2: 音声合成エンジン (Audio) の統合
* 記事内容を要約・台本化するLLMプロンプトの設計。
* Text-to-Speech API（Google Cloud TTS、VOICEVOX、OpenAI TTSなど）との連携。
* BGMの自動ミキシングと音声ファイルの生成。

### Phase 3: 動画合成エンジン (Video) の統合
* 音声のタイミングに合わせた自動スライド（画像・グラフ）生成。
* FFmpeg等を用いた、音声＋画像の動画エンコード処理。
* 将来的にはアバター動画生成API（HeyGen, D-ID等）との連携。

### Phase 4: 管理ダッシュボード (Web UI)
* 生成された記事、音声、動画をプレビュー・編集・公開管理するためのNext.jsアプリケーション。
* YouTube, TikTok, 各種ブログメディア（WordPress等）への自動投稿・配信管理機能。
